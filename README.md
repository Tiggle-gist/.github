# Tiggle 🤖

> 자연어 명령 하나로 군집 마이크로봇이 현실에서 움직이는 시스템

영화 빅히어로의 마이크로봇에서 영감을 받아, 자연어 instruction 하나로 템플릿 없이 최적의 행동을 생성하고 군집 로봇이 이를 현실에서 수행하는 시스템을 만들고 있습니다.
